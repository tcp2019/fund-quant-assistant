# v2.7 组合每日盈亏曲线 设计规格

> **Status:** active
> **Created:** 2026-06-28
> **Parent spec:** [2026-06-27-nav-daily-change-design.md](./2026-06-27-nav-daily-change-design.md)

## 目标

总览页展示近 30/90 个交易日组合**每日盈亏**折线，与 v2.5「今日盈亏」公式一致，帮助回答「最近组合涨跌怎么样」。

## 公式

```
单只日盈亏(日 t) = 份额 × (当日净值 − 该基金上一交易日净值)
组合日盈亏(日 t) = Σ 各持仓当日可算日盈亏
```

份额取自最新快照（不追溯历史申赎）。

## 范围

### 包含

1. `build_daily_history(days)` — 按基金自身相邻净值日聚合
2. `GET /api/portfolio/daily-history?days=30|90`
3. 总览次屏折线图 + 30/90 天切换
4. `complete` 标记：当日是否全部有份额持仓都贡献了日盈亏

### 不包含

- 累计盈亏曲线、市值曲线主展示
- 按快照还原历史份额
- WebSocket 推送

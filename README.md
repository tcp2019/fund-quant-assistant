# 基金持仓管家（Fund Quant Assistant）

个人基金持仓管理与分析 Web 工具。上传持仓截图导入基金，看清组合状态，并获得**可解释的调仓建议**（非投资建议）。

> **免责声明：** 本产品仅供个人学习与持仓管理，**不构成任何投资建议**。所有建议均为基于历史数据的规则化输出，使用前请自行判断风险。

## 功能

- **截图 OCR 导入** — 支付宝、天天基金、理财通持仓截图
- **组合总览** — 实时市值、今日盈亏、大类配置是否均衡
- **本周建议** — 建议卖出 / 增配 / 新买，合并原「机会」与「信号」
- **持仓明细** — 明细列表与快照历史
- **深入了解** — 风险指标、热点主题、回测模拟（进阶）
- **三层规则引擎** — 配置再平衡 + 风险集中度 + 业绩质量
- **本地单用户部署** — SQLite 存储，数据留在本机

## 环境要求

- Python 3.11+
- Node.js 18+（推荐 20+）
- npm

可选：安装 `[ocr]` 依赖以启用 PaddleOCR 本地识图。

## 安装与启动

见仓库内 `backend/`、`frontend/` 说明。项目根目录：

```bash
./scripts/dev.sh
```

| 服务 | URL |
|------|-----|
| 前端 | http://localhost:5173 |
| 后端 API | http://127.0.0.1:8000 |

## 页面

| 路由 | 说明 |
|------|------|
| `/` | 总览 |
| `/advice` | 本周建议 |
| `/holdings` | 持仓 |
| `/settings` | 设置 |
| `/insights` | 深入了解（风险 / 热点 / 回测） |
| `/import` | 导入持仓（无主导航，从总览或持仓进入） |

旧路由 `/signals`、`/opportunities`、`/analysis` 会自动跳转到新页面。

## 测试

